0~n-1中缺失的数字

一个长度为n-1的递增排序数组中的所有数字都是唯一的，并且每个数字都在范围0～n-1之内。在范围0～n-1内的n个数字中有且只有一个数字不在该数组中，请找出这个数字。

二分法，注意找到的是第一个index<nums[index]的位置，当没有这种情况存在时，找到的是数组末尾的index，所以当返回的是数组末尾是，需要判断nums[index]到底是index还是index+1
