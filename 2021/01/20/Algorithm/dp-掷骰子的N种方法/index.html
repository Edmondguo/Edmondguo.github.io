掷骰子的N种方法

这里有 d 个一样的骰子，每个骰子上都有 f 个面，分别标号为 1, 2, …, f。

我们约定：掷骰子的得到总点数为各骰子面朝上的数字的总和。

如果需要掷出的总点数为 target，请你计算出有多少种不同的组合情况（所有的组合情况总共有 f^d 种），模$10^9 + 7$后返回。

DP

一般来说组合问题，如果是要求所有可能的方案都是用递归，但是如果要求方案总数就是要用动态规划。

dp数组中存储的是(当前要投掷第几个筛子, 当前投掷过的所有筛子的总和)。一般来说，是dp[i][j] = sum(dp[i - 1][j - k:j])（没有做特殊判断）。这道题需要多留意边界判断，有效的边界判断不仅可以避免数组越界还可以减少搜索空间。
